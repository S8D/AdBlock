# AdBlock for DD-WRT / Android / iOS base on Linux
AdBlock hosts scripts for Linux Platform

Scrips này chạy được nên firmware dd-wrt / iOS / Linux hay thậm chí Android

**Yêu cầu:**
- cURL
- chmod +x [Tên file].sh

**Sử dụng:** sh [s.sh](./s.sh)

**iOS:** sh [i.sh](./i.sh)
